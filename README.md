# django-career-portal
A Django-based career portal that enables candidates to register, log in, and apply for job opportunities, while providing HR teams with a centralized dashboard to manage and review applications. The system uses JWT (JSON Web Token) authentication to ensure secure, stateless user sessions and role-based access control.

🚀 Features
Candidate registration and login system
Secure authentication using JWT tokens
Job application submission and tracking
HR dashboard to view and manage candidate applications
Role-based access (Candidate & HR)
RESTful API integration 

🛠️ Tech Stack
Backend: Django / Django REST Framework
Authentication: JWT (JSON Web Tokens)
Database: (SQLite / PostgreSQL)

🔐 Security
Implements JWT-based authentication for secure and scalable session management, reducing dependency on traditional session-based login systems.
